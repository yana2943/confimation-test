# お問い合わせフォーム

## 環境構築
Dockerビルド
1. docker-compose.ymlを作成
2. docker compose up -d --build

Laravel環境構築
1. docker-compose exec php bash
2. composer インストール
3. php artisan migtate
4. php artisan db seed
